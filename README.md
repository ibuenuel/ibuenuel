## Featured Projects

### [CardBase.ch](https://cardbase.ch) — Swiss TCG Marketplace
> The first dedicated Swiss online marketplace for Trading Card Games

- Serving **local collectors** across Switzerland
- Built with **Python 3.12, Django 6.0, PostgreSQL** on custom VPS infrastructure
- Automated data pipeline syncing **100,000+ cards** across 6 major TCGs via REST APIs & Celery
- Secure payments via **Stripe Connect** with escrow-based wallets & automated fee calculation
- Integrated **Swiss Post logistics** with multilingual support (DE, EN, FR, IT)

---

### [dotnet-azure-starter](https://github.com/ibuenuel/dotnet-azure-starter) — .NET 10 Boilerplate
> Production-ready ASP.NET Core 10 boilerplate with Clean Architecture, Azure deployment, Docker, and CI/CD

- **Clean Architecture** with four strict layers — dependency rules enforced by architecture tests at build time
- **Result Pattern** instead of exceptions for business failures — typed `Result<T>` with structured error values
- **72 tests** across unit, architecture (NetArchTest), and integration (Testcontainers + real SQL Server)
- **One-command Azure deployment** via Bicep — App Service, SQL Database, Key Vault, App Insights, ACR
- **Full CI/CD** via GitHub Actions — CI on every PR, CD to Azure App Service via Docker + ACR on push to `main`
- **Docker Compose** full-stack local development — API + SQL Server 2022, auto-migrations and seed data on first start
